Weather Forecast Application

This project is a simple web-based Weather Forecast Application that provides current and forecasted weather details for a specified city. It also features interactive charts and a chatbot interface for enhanced user experience.

Features

- Get the current weather and a 5-day forecast for any city.
- Switch between temperature units (Celsius/Fahrenheit).
- Paginated forecast display.
- Interactive charts using Chart.js (bar chart, pie chart, line chart) for visual representation of temperature and weather conditions.
- A filter section to interact with the forecast data:
  - Show only rainy days.
  - Sort forecasted temperatures in ascending or descending order.
  - Find the highest temperature day.
- Simple chatbot interface for asking weather-related questions.

Built With

- HTML5: Structure of the application.
- CSS3: Styling for layout and responsiveness.
- JavaScript: Core functionality for fetching weather data, displaying charts, and chatbot functionality.
- Chart.js: A JavaScript library for displaying charts and graphs.


### Prerequisites

To run this application locally, all you need is a web browser.
